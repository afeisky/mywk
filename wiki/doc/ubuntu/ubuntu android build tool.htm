--------------------------------------------------------
1）    安装samba，ubuntu命令行下输入sudo apt-get install samba
2）    配置samba，ubuntu命令行下输入 sudo gedit /etc/samba/smb.conf，在打开文件的最后，输入共享文件信息，如下：
[Samuel]     #共享的文件夹名称
comment = Shared Folder with username and password
path = /home/wchaofei         #共享的文件夹目录
public = yes
writable = yes
force create mode = 0777
force directory mode = 0777
force user = wchaofei  #nobody
force group = nogroup
available = yes
guest ok = no       #加了这句，就表示要用户名及密码才能登陆()，去掉这句，所有人都能不用密码登陆
browseable = yes
3)    增加个用户 sudo smbpasswd -a username              #add a user.
4）    保存以上信息后，重启samba。 sudo /etc/init.d/samba restart          #restart samba
4）    在windows 端输入linux机器的ip地址，就可以看到以上共享文件夹”Samuel”。
