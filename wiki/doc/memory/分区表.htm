System Partition

1. MTK 默认的system分区大小为1.5G

2. MP版本编译的system.img大小为661MB(32bit,不采用odex优化),而app、private-app、framework下作odex优化预留200MB空间，算900MB

3.GMS导入到system分区，GMS必需app总共325MB，GMS所有app共662MB(32bit下)。给予600MB预留空间

4. 第三方apk导入，目前第三方apk及其库文件为236+25=261MB,预留400MB空间

5.MTK升级预留200MB空间

6.后期维护预留200MB空间

7.写保护需要size对齐8M，加4M

   

Android Size = 900MB +  600MB + 400MB + 200MB + 200MB + 4MB= 2304MB
